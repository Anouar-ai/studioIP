# **App Name**: StreamWise IPTV Guide

## Core Features:

- Provider Comparison Table: Display a sortable table comparing IPTV providers based on channels, VOD content, price, and ratings, similar to the one provided in the article.
- IPTV Setup Guide: Provide a step-by-step guide with instructions on how to set up IPTV on various devices, as described in the original request.
- Content Summarization: AI tool to summarize IPTV article for a brief overview using a Large Language Model.
- Dynamic Content Updates: Use LLM to scan for the best current links. Then store in-memory. And deliver current top options.

## Style Guidelines:

- Primary color: Deep Blue (#1E3A8A) to convey trust, reliability, and a sense of technology.
- Background color: Very light gray (#F9FAFB), nearly white, for a clean and modern look.
- Accent color: A vibrant orange (#EA580C) to highlight key information, calls to action, and important UI elements. This can represent the future-oriented angle of the application.
- Headline font: 'Poppins', a geometric sans-serif for a precise and contemporary look.
- Body font: 'Inter', a grotesque-style sans-serif for a modern, neutral feel that is highly readable.
- Use modern, flat icons to represent different IPTV providers, content types, and device types.
- A clean, organized layout with clear headings and subheadings to enhance readability and user experience.
